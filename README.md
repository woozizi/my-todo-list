# 간단한 할 일 리스트 App

## 기술 스택

- **Next.js App Router (14+)**
- **TypeScript**
- **Tanstack Query v5**
- **Tailwind CSS**
- **json-server**
- **ESLint, Prettier**

## 주요 기능

- **투두 CRUD** (추가, 수정, 삭제, 완료 토글)
- **완료/전체/남은 필터** (드롭다운, 실시간 개수 표시)
- **SSR + Hydration** (React Query v5 `prefetchQuery` + `HydrationBoundary`)
- **반응형 UI** (모바일~데스크톱 대응)
